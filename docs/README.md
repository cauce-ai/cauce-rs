## Contributing to Documentation

Documentation follows the same contribution process as code. See [CONTRIBUTING.md](../CONTRIBUTING.md) for details.

When adding documentation:

1. Place files in the appropriate subdirectory
2. Update this README with links to new documents
3. Use GitHub-flavored Markdown
4. Use Mermaid for diagrams (GitHub renders natively)
